# Sound Files Required

This directory should contain the following MP3 audio files for game sound effects:

| Filename             | Description                            | Duration | Volume |
| -------------------- | -------------------------------------- | -------- | ------ |
| `train-moving.mp3`   | Train wheel/movement sounds, loopable  | ~2s      | 30%    |
| `train-stopping.mp3` | Train brake sounds                     | ~1s      | 40%    |
| `door-close.mp3`     | Train door closing sound               | ~0.5s    | 50%    |
| `announcement.mp3`   | Ding sound before station announcement | ~1s      | 60%    |
| `seat-click.mp3`     | Subtle tap sound for seat interaction  | ~0.2s    | 20%    |
| `seat-claim.mp3`     | Success sound when claiming a seat     | ~0.5s    | 50%    |
| `win-jingle.mp3`     | Celebratory jingle for winning         | ~2s      | 60%    |
| `lose-sound.mp3`     | Sympathetic sound for losing           | ~1.5s    | 50%    |
| `npc-grab.mp3`       | Quick grab sound when NPC takes seat   | ~0.3s    | 40%    |

## Notes

- All files should be in MP3 format for browser compatibility
- Files should be appropriately compressed for web use
- Consider using royalty-free audio clips that evoke Mumbai local train atmosphere
- The game is fully playable without these files (sounds will fail silently)
